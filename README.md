This project (made with UIKit) features:<br>

**Alamofire**;<br>
**Certificate pinning**;<br>
**CompositionalLayout**;<br>
**Crashlytics**;<br>
**DispatchGroup**;<br>
**Firebase Analytics**;<br>
**Hero Animation**;<br>
**Light and dark mode**;<br>
**Localized strings**;<br>
**MVVM-Coordinator** (view code);<br>
**Protocol oriented**;<br>
**Singleton**;<br>
**Snapshot tests**;<br>
**Unit tests**.<br>
